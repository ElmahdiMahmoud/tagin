Tagin

jQuery plugin for managing tags

Examples

Markup

Set an input field type="text" and give it an id="YOUR-ID" and leave the rest on tagin! 

<input type="text" id="WHATEVER" placeholder="Add tags" />

Default

Default option: comma , key 

$("SELECTOR").tagin();

Maximum Tags

maxtag option allows you to control the maximum number of tags that users can add. 

$("SELECTOR").tagin({ maxTags: 5 });

Type: Number
Default: null

Backspace (del)

backSpace option remove tags. You can also use [Delete] button on your keyboard. 

$("SELECTOR").tagin({ backSpace: true });

Type: Boolean
Default: false

Edit Mode

Double click on the tag you want to edit. 

$("SELECTOR").tagin({ editMode: true });

Type: Boolean
Default: false
